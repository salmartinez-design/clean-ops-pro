**Step 2 — Fix the calculation:**

The upsell price must NOT make an async API call. It must calculate synchronously on the client using the pricing data already loaded for the booking widget.

The widget already has the pricing tiers loaded (it uses them to show the quote price). Reuse that same data:

```javascript
// The widget already has pricingData loaded
// Use it directly — no API call needed

const getUpsellRate = (sqft, cadence, pricingData) => {
  // Find the recurring scope tiers
  const recurringScope = pricingData.scopes.find(s => 
    s.name.toLowerCase().trim() === 'recurring cleaning'
  );
  
  if (!recurringScope) return null;
  
  // Find matching tier for this sqft
  const tier = recurringScope.tiers.find(t => 
    sqft >= t.sqft_min && sqft <= t.sqft_max
  );
  
  if (!tier) return null;
  
  // Apply cadence multiplier
  const multipliers = {
    'weekly': 0.80,
    'biweekly': 0.90,
    'every_4_weeks': 1.00
  };
  
  const multiplier = multipliers[cadence] || 1.00;
  const recurringRate = tier.base_price * multiplier;
  const firstVisitRate = recurringRate * (1 - (offerSettings.upsell_discount_percent / 100));
  
  return { recurringRate, firstVisitRate };
};
```

Call this synchronously when cadence is selected — no loading state needed, no API call, instant result.
