THE FIX — RECURRING HOURLY RATES FOR PHES:
Currently the recurring calculation uses:

recurring_price = base_price × frequency_multiplier


For Phes this must instead use three separate hourly rates by frequency:

Weekly:        estimated_hours × $60.00
Biweekly:      estimated_hours × $65.00  
Every 4 Weeks: estimated_hours × $70.00


Where estimated_hours is derived from the same sqft tier lookup already used for recurring cleaning.

Step 2 — Update the recurring price calculation function:
In the pricing calculation logic, when calculating recurring cleaning price:

const calculateRecurringPrice = (scope, sqftTier, cadence) => {
  
  // If company uses frequency hourly rates (Phes)
  if (scope.use_frequency_hourly_rates) {
    const hourlyRates = {
      'weekly':       scope.weekly_hourly_rate,    // $60
      'biweekly':     scope.biweekly_hourly_rate,  // $65
      'every_4_weeks': scope.monthly_hourly_rate,  // $70
    };
    
    const hourlyRate = hourlyRates[cadence] || scope.monthly_hourly_rate;
    
    // Derive estimated hours from the tier
    // base_price in the tier was calculated at $65/hr base
    // So estimated_hours = tier.base_price / 65
    const estimatedHours = sqftTier.base_price / 65;
    
    const price = estimatedHours * hourlyRate;
    
    // Apply minimum bill if set
    const minimum = scope.minimum_bill || 0;
    return Math.max(price, minimum);
  }
  
  // Otherwise use multiplier system (all other tenants)
  const multipliers = {
    'weekly':       0.80,
    'biweekly':     0.90,
    'every_4_weeks': 1.00
  };
  
  const multiplier = multipliers[cadence] || 1.00;
  return sqftTier.base_price * multiplier;
};


Step 3 — Update the upsell card calculation:
The upsell card currently uses the hardcoded synchronous tiers with multipliers. Update to use the same calculateRecurringPrice function:

const calculateUpsellPrice = (sqft, cadence, scope, discountPercent = 15) => {
  if (!sqft || sqft <= 0) return null;

  // Find matching tier
  const tier = scope.tiers.find(t => sqft >= t.sqft_min && sqft <= t.sqft_max);
  if (!tier) return null;

  // Use frequency hourly rates for Phes
  const hourlyRates = {
    'weekly': 60.00,
    'biweekly': 65.00,
    'every_4_weeks': 70.00
  };

  const hourlyRate = hourlyRates[cadence] || 70.00;
  const estimatedHours = tier.base_price / 65; // derive hours from $65/hr base tier
  const recurringRate = Math.round(estimatedHours * hourlyRate * 100) / 100;
  const firstVisitRate = Math.round(recurringRate * (1 - discountPercent / 100) * 100) / 100;

  return { recurringRate, firstVisitRate };
};
