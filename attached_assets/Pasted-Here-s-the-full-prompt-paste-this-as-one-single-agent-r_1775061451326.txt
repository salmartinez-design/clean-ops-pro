- Create lead record: status = booked, source = online_booking, booked_at = now

- Fire internal office notification immediately: "Very Dirty callback — [Name] — [phone] — [address]. Needs office contact."

## SECTION 3 — LEAD PIPELINE UI

**Route:** `/leads`

**Page header:**
- Title: "Lead Pipeline"
- Top right: "+ Add Lead" button — opens a slide-over drawer to manually create a lead with all fields

**Status filter pills (horizontal row, toggleable):**
Each pill shows the status name and a live count of leads in that status. Clicking toggles that status on/off in the table. All active by default.

Pill order and colors:
- Needs Contacted — red
- Contacted — yellow
- Quoted — blue
- Follow Up — orange
- Booked — green
- Closed — gray (unchecked by default)

**Table columns (in this exact order):**
Name | Address | Zip | Source | Status | Created | Last Contacted | Quoted | Booked | Assigned To | Agreement | Actions

Design rules:
- Name column: First name + Last name as a clickable link — opens lead detail drawer
- Source column: render as a small pill badge, not plain text. Color-code by source: Google Local Services = blue, Google Search = light blue, Facebook = indigo, Referral = green, Realtor = teal, Online Booking = mint, Manual = gray
- Status column: render as colored pill matching the filter pill colors above
- Last Contacted: show relative time ("2 days ago") with full date on hover
- Quoted column: show dollar amount if quote exists, dash if not
- Booked column: show date if booked, dash if not
- Assigned To column: show assignee name or "Unassigned" in muted red if empty
- Agreement column: show a small checkmark icon if signed, dash if not
- Actions column: "View" button only — all actions happen inside the lead detail drawer

**No data state:**
If no leads exist for the active filters, show: "No leads found for the selected filters." — do not show an empty table frame.

**Pagination:** 25 records per page, with total count shown ("Showing 1–25 of 47 leads")

**Search bar:** searches across name, email, phone, address, zip in real time

**Advanced filters (collapsed by default, "Show Advanced Filters" toggle):**
- Date range (Created)
- Source (multi-select)
- Assigned To (multi-select)
- Scope (multi-select)

## SECTION 5 — SEQUENCE WIRING

Connect the existing follow_up_sequences engine to the lead pipeline:

- Abandoned booking → enroll lead in Sequence A (quote_followup) automatically
