DRAG AND DROP BEHAVIOR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Use @dnd-kit/core for all drag and drop.

DRAG JOB CHIP along employee row:
  → Changes scheduled_time
  → Snaps to nearest 30-minute increment
  → Show ghost chip at drop position
  → On drop: PATCH /api/jobs/:id 
    { scheduled_time: new_time }
  → Show toast: "Job rescheduled to 10:00 AM"

DRAG JOB CHIP to different employee row:
  → Reassigns to that employee
  → On drop: PATCH /api/jobs/:id 
    { assigned_to: new_employee_id,
      scheduled_time: new_time }
  → Show toast: "Job reassigned to [name]"

DRAG from UNASSIGNED PANEL onto employee row:
  → Assigns the job to that employee at that time
  → On drop: PATCH /api/jobs/:id
    { assigned_to: employee_id,
      scheduled_time: time }
  → Job disappears from unassigned panel
  → Chip appears on board in SCHEDULED state
  → Show toast: "Job assigned to [name]"

DRAG CONSTRAINTS:
  → Cannot drag complete jobs
  → Show visual indicator (reduced opacity + 
    "not-allowed" cursor) when hovering over 
    a time slot that conflicts with another job 
    for the same employee
