**REPLIT PROMPT — DATE PICKER DEFAULT + LEAD TIME SETTING:**

**Fix 1 — Default date selection:**

When Step 4 (Date picker) loads, automatically pre-select the first available bookable date that is at least `minimum_lead_days` from today. Do not default to today or tomorrow — default to 7 days out unless a setting overrides it.

```javascript
const getDefaultDate = (minimumLeadDays = 7) => {
  const today = new Date();
  const defaultDate = new Date(today);
  defaultDate.setDate(today.getDate() + minimumLeadDays);
  
  // If default date falls on Sunday (0) or Saturday (6 — if Saturday unavailable):
  // Advance to next Monday
  while (defaultDate.getDay() === 0) {
    defaultDate.setDate(defaultDate.getDate() + 1);
  }
  
  return defaultDate;
};
```

Pre-select this date when the calendar renders. The customer can change it — this is just the starting selection so they don't land on a blank calendar.

Also advance the calendar view to the month containing the default date automatically. If today is March 29 and default is April 5, open the calendar to April, not March.

**Fix 2 — Disable dates within lead time window:**

Disable (gray out, unclickable) all dates that are fewer than `minimum_lead_days` from today. Disable all Sundays. Keep Saturdays enabled (Phes works Saturdays 9am–12pm).

```javascript
const isDateDisabled = (date, minimumLeadDays) => {
  const today = new Date();
  today.setHours(0, 0, 0, 0);
  
  const minDate = new Date(today);
  minDate.setDate(today.getDate() + minimumLeadDays);
  
  // Disable if before minimum lead time
  if (date < minDate) return true;
  
  // Disable Sundays
  if (date.getDay() === 0) return true;
  
  return false;
};
```

**Fix 3 — Add booking lead time to Company Settings:**

In Company Settings → a new "Booking Rules" section (or add to existing General Settings tab), add:

**Field: Minimum Booking Lead Time**
- Label: "Minimum days before a booking can be scheduled"
- Input: number input with "days" suffix
- Default: 7
- Range: 1–30
- Help text: "Customers cannot book a cleaning fewer than this many days from today. The calendar will default to this many days out when a customer reaches the date selection step."
- Maps to: `booking_lead_days` in a `booking_settings` table per company

**Fix 4 — booking_settings table:**

```sql
CREATE TABLE IF NOT EXISTS booking_settings (
  id SERIAL PRIMARY KEY,
  company_id INTEGER NOT NULL UNIQUE REFERENCES companies(id) ON DELETE CASCADE,
  booking_lead_days INTEGER NOT NULL DEFAULT 7,
  booking_advance_days INTEGER NOT NULL DEFAULT 60,
  -- booking_advance_days = how far in advance customers can book (default 60 days)
  available_days JSONB NOT NULL DEFAULT '{"sun": false, "mon": true, "tue": true, "wed": true, "thu": true, "fri": true, "sat": true}',
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Seed defaults for all existing companies
INSERT INTO booking_settings (company_id)
SELECT id FROM companies
ON CONFLICT (company_id) DO NOTHING;
```
