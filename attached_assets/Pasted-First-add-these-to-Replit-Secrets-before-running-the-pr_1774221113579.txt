## PART 3 â€” SYNC ENGINE

Create services/quickbooks-sync.ts

Qleno is always the source of truth.

**syncCustomer(companyId, customerId)**
- Check qb_customer_map for existing QB customer ID

  TxnDate, DueDate, Line items (each line item as
  SalesItemLineDetail), GlobalTaxCalculation
- If existing: update invoice in QB using SyncToken
  (handles tip additions, hour adjustments, discounts
  added after initial creation â€” all done in Qleno,
  pushed to QB)
- Save qb_entity_id to qb_sync_queue

**syncPayment(companyId, invoiceId)**
- Only called when invoice status = 'paid'
- Creates a Payment record in QB linked to the invoice
  Fields: CustomerRef, TotalAmt, TxnDate,
